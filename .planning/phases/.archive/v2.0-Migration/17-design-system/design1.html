Motion Engine

"What does it mean?" It means orchestrating how pixels move so the app feels alive, not static.

Framer Motion REACTIVE

Use for: Layout transitions, dragging elements, and complex entry sequences. It handles the "logic" of movement.

CSS Transitions MICRO

Use for: Hover states, button scales, and color shifts. Zero performance overhead.

Lottie NARRATIVE

Use for: Celebratory "Success" animations or complex empty-state illustrations.

Form Logic ↔ UI

The Connection: React-Hook-Form + Zod
VALIDATION TRIGGERED
The Bridge

Recommendation: Logic (Zod) validates the data; the UI (Shadcn Form) listens for errors. If "CaseNumber" is invalid, the Logic tells the UI to turn the Slab red and shake it.

Skeleton Systems

Concern: Design System (Primitive) + Page Level (Layout)

SYSTEM: Define the "SlabSkeleton" component in your design system.
PAGE: Arrange those skeletons on the page to mirror the final content layout. This prevents "Layout Shift."
